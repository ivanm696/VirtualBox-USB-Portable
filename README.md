# VirtualBox USB Portable

Автоматическая установка VirtualBox прямо на USB флешку — запускай Windows 11 с флешки на любом ПК.

## Быстрый старт

1. Вставь флешку (64+ GB, USB 3.0, NTFS)
2. Правая кнопка на `УСТАНОВИТЬ_НА_ФЛЕШКУ.bat` → **Запуск от имени администратора**
3. Введи букву флешки (например: `E`)
4. Дождись установки
5. Скачай [Windows 11 ISO](https://www.microsoft.com/software-download/windows11) → скопируй в `E:\ISO\`
6. Запусти `START_VirtualBox.bat`

## Что делает установщик

- ✅ Скачивает VirtualBox **7.1.6** с официального сайта
- ✅ Устанавливает прямо на флешку через `msiexec`
- ✅ Проверяет свободное место (минимум 10 GB)
- ✅ Создаёт конфигурацию с путями на флешку
- ✅ Создаёт `START_VirtualBox.bat` для запуска

## Требования

| Параметр | Минимум |
|---|---|
| Флешка | 64 GB, USB 3.0, NTFS |
| Свободное место | 10 GB (для VirtualBox) + 50+ GB для VM |
| ОС хоста | Windows 10/11 |
| Права | Администратор |
