СОДЕРЖИМОЕ АРХИВА:
==================

📄 УСТАНОВИТЬ_НА_ФЛЕШКУ.bat
   └─ Запускайте ЭТО первым! (от имени Администратора)
      Автоматически запустит PowerShell установщик.

📄 Install_VirtualBox_USB.ps1
   └─ PowerShell скрипт который:
      ✓ Создаст структуру папок на флешке
      ✓ Скачает VirtualBox с официального сайта
      ✓ Установит VirtualBox прямо на флешку
      ✓ Создаст конфигурацию
      ✓ Создаст файл запуска ЗАПУСК_VirtualBox.bat

📄 Инструкция_VirtualBox_USB.pdf
   └─ Подробная пошаговая инструкция с таблицами
      Читайте если что-то непонятно!

КАК НАЧАТЬ:
===========

1. Вставьте флешку (минимум 64 GB, USB 3.0, формат NTFS)

2. Правая кнопка на УСТАНОВИТЬ_НА_ФЛЕШКУ.bat
   → "Запуск от имени администратора"

3. Введите букву флешки (например: E)

4. Дождитесь завершения установки

5. Скачайте Windows 11 ISO с:
   https://www.microsoft.com/ru-ru/software-download/windows11

6. Скопируйте ISO в папку E:\ISO\ на флешке

7. Запустите E:\ЗАПУСК_VirtualBox.bat

8. Создайте ВМ и установите Windows 11
   (подробно в PDF инструкции)

ТРЕБОВАНИЯ:
===========
• Windows 10/11 на основном ПК
• Права Администратора
• Интернет (для скачивания VirtualBox ~100 MB)
• Флешка: 64+ GB, USB 3.0, NTFS
